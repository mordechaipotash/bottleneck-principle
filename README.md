# The Bottleneck Principle

> "The bottleneck isn't AI capability anymore. It's human reception."

A framework for human sovereignty in the age of AI.

---

## The Core Thesis

Somewhere between GPT-3.5 and Claude 3, something shifted. AI capability stopped being the constraint.

The new bottleneck: **Can humans understand enough to decide with confidence?**

Everyone thinks the solution is making AI more human-like. More autonomous. More intelligent.

Wrong direction. The solution isn't smarter AI. It's better translation.
